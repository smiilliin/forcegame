# Forcegame
Tension force game

![screenshot](https://github.com/user-attachments/assets/6debb2db-2acc-4957-9260-66c05dea4db4)

## How to play
- Click or touch to join a pointer  
- Shift key to dash
## Install
Install typescript packages
```bash
npm install
```
## Run development server
Run webpack dev server
```bash
npm run dev
```
## Build bundle.js
Build bundle file
```bash
npm run build
```
